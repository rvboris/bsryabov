# Personal CV page

Simple static website build with gulp, nunjunks template language and data model in JSON, support PWA and service workers precache.

 - Gulp
 - Webpack for javascript
 - Postcss
 - Nunjucks
 - Browsersync

### NPM tasks

Develop mode with browsersync

```
npm run dev
```

Build for production

```
npm run build
```

Lint all

```
npm run lint
```
